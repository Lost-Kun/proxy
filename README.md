# proxy
精简的Node代理服务器

特点：多进程、控制台彩色输出

应用场景：局域网内部系统的代理转发

使用步骤：

         （1）下载项目到本地，npm install 下载依赖包

         （2）可修改config.js下项目启动端口

         （3）node app.js启动项目

         （4）Internet选项的局域网设置中配置代理（目前只能代理http请求）
